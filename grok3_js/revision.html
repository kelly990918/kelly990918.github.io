1. Variables (變數)

What: Containers to store data.

Keywords: let (can change), const (can’t change).

Example:

let name = "Kelly"; // String (字串)
const age = 26;     // Number (數字)

Your Exercise:

let name = "Kelly";
const age = 26;

Result: Worked perfectly—stored your name and age!

2. Data Types (資料類型)

What: Kinds of data you can store.

Types:

  • String (字串): Text in quotes ("hello")
  • Number (數字): Numbers, no quotes (42)
  • Boolean (布林值): true or false
  • Array (陣列): List in []
    • Min: 0 items ([])
    • Max: ~4.29 billion
  • Object (物件): Key-value pairs in {}

Example:

let color = "Navy";
let year = 1999;
let isLovePizza = false;
let movies = ["Transformers", "Dune"];
let person = {firstName: "Kelly"};

Your Exercise:

let color = "Navy";
let year = 1999;
let isLovePizza = false;
let movies = ["Transformers", "Star Wars", "Dune"];
let name = {firstName: "Minerva", lastName: "Louie"};

Result: Almost perfect! Reused name (conflict)—use unique names like person.

3. Operators (運算子)

What: Symbols for math or comparison.

Types:

  • Arithmetic: +, -, *, /
  • Comparison: ==, !=, >, <, >=, <=
  • Modulo (%): Remainder (e.g., 5 % 2 = 1)

Example:

let x = 5;
let y = 10;
console.log(x + y); // 15

Your Exercise:

let x = 5;
let y = 10;
let sum = x + y;       // 15
let difference = x - y; // -5
let product = x * y;   // 50
let quotient = x / y;  // 0.5
let isEqual = x == y;  // false
let isLess = x < y;    // true

Result: Perfect—covered all operators!

4. Control Structures (控制結構)

4.1 Conditional Statements (條件語句)

Keywords: if, else if, else

Syntax:

if (condition) { ... } else if (condition) { ... } else { ... }

Example:

let number = 5;
if (number > 0) {
    console.log("Positive");
} else if (number === 0) {
    console.log("Zero");
} else {
    console.log("Negative");
}

Your Exercises:

Positive/Negative/Zero:

if (number > 0) {
    console.log("The number is positive.");
} else {
    console.log("The number is negative.");
} if else {
    console.log("The number is zero.");
}

Weather Checker:

let temperature = 40;
if (temperature > 30) {
    console.log("It's hot!");
} else if (temperature < 15) {
    console.log("It's cold!");
} else {
    console.log("It's nice!");
}

Age Group:

let age = 20;
if (age > 20) {
    console.log("Adult");
} else if (age < 13) {
    console.log("Kid");
} else {
    console.log("Teen");
}

Even or Odd:

let number = 4;
if (number % 2 == 0) {
    console.log("Even");
} else {
    console.log("Odd");
}

Traffic Light:

let color = "red";
if (color === "red") {
    console.log("Stop");
} else if (color === "green") {
    console.log("Go");
} else if (color --- "yellow") {
    console.log("Wait");
} else {
    console.log("Invalid");
}

Your Mistakes:

  • Wrong Syntax: if else instead of else if.
  • Range Error: age > 20 missed 20—use >= 20.
  • Typo: color --- "yellow" should be ===.

Lesson: Use else if, check ranges, avoid typos.

4.2 Loops (迴圈)

Keyword: for

Syntax: for (start; condition; step) { ... }

Example:

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Your Exercise:

for (let i = 1; i < 10; i++) {
    console.log(i); // 1 to 9
}

Updated Attempt:

for (let i = 1; i < 11; i++) {
    console.log(i); // 1 to 10
}

Result: Fixed to include 10!

Bigger Number:

    if (a > b) {
        return a;
    } else {
        return b;
    }
}

let f = 8;
let compare = bigger(e, f);

  • Syntax: let "books", .push =, conole.
  • No Print: Missed console.log.
  • String vs. Boolean: "true" instead of true.

Lesson: Check syntax, print results, use correct types.

7. Properties & Methods (屬性與方法)

String Tools

  • .length: 
    console.log("cat".length); // 3
  • .toLowerCase(): 
    console.log("CAT".toLowerCase()); // "cat"
  • .toUpperCase(): 
    console.log("cat".toUpperCase()); // "CAT"
  • .includes(text): 
    console.log("hello".includes("lo")); // true
  • .indexOf(text): 
    console.log("banana".indexOf("a")); // 1
  • .slice(start, end): 
    console.log("hello".slice(0, 2)); // "he"

Array Tools

  • .length: 
    console.log(["a", "b"].length); // 2
  • .push(item): 
    let arr = []; arr.push("x"); // ["x"]
  • .pop(): 
    let arr = ["x", "y"]; arr.pop(); // "y"
  • .includes(item): 
    console.log([1, 2].includes(2)); // true

Tips for Revising

  • Practice: Test in the console or .html files.
  • Mistakes to Watch: Syntax, ranges, data types.
